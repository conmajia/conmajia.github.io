    Generate URLs with Search Keywords#

    To crawl a website, a correct resource location is required to fetch web pages containing desired contents. In this Ebay example, a URL1 with a keyword in the path parameter is used to obtain pages returned by the server based on the keywords to be searched.

    https://www.ebay.com/sch/i.html?_nkw=web+crawler

    This shall return a page as follow.

    Now, a function wrapping keyword strings into a target URL is able to be established.

    def buildUrl(baseUrl, param, keyword):
    if (keyword):
        return "{}?{}={}".format(baseUrl, param, keyword.replace(" ", "+"))
    else:
        return baseUrl
url = buildUrl(
    "https://www.ebay.com/sch/i.html",
    "_nkw",
    "web crawler"
)

    It is easy to write a regular expression to extract the literal book title. Be aware of that the source text may not be always well-formatted, thus conditional operators "? and .*? are used to guarantee the expression matches in as many as potential situations.

    <h3 class="?s-item__title.*?"?>(.*?)<\/h3>

    Which matches book title in the first matching group.

    import re
pattern = '<h3 class="?s-item__title.*?"?>(.*?)<\/h3>'
match = re.search(pattern, response.text)
print(match.group(1))

    Structural Solutions#

    Since regular expression is so complicated, time-consuming and scenario-dependent, even the most experienced masters may sometimes feel it arduous to build proper expressions. Based on the fact of tree-structured DOM that elements of an HTML file are organized, several decent DOM orientated content analyzers/parsers could help to reduce the difficulty and complexity in data extraction.

    XPath and JSONPath, as the names suggest, are libraries for discovery nodes through specified paths in XML and JSON files, respectively. Beautiful Soup is another powerful “all-in-one” parser for HTML/XML content extraction, depending on the parsers selection.

    Let’s try the Ebay example again. This time, the Beautiful Soup library will be used instead.

    from bs4 import BeautifulSoup
bs = BeautifulSoup(response.text, "html.parser")

    As analyzed in figure 3, information of an item is represented by contents contained in DOM nodes in HTML source. Skimming through the DOM, we will get a class-marked structure of item information as in figure 4.

    Regardless of deeply nested in the source, it is possible to instantly extract parts (nodes) that contains information we need using the Beautiful Soup library. In the following example, an html.parser, which is the Python’s standard parser, is assigned to the instance to cope with HTML source inputs.

    from bs4 import BeautifulSoup
bs = BeautifulSoup(response.text, "html.parser")
bs.title.string

    With the power of DOM, we are able to search in parsed DOM tree by simply passing tag names and classes as parameters to the .find() function. The first parameter of .find() is a tag name of the element to search, and a later class_2 parameter restricts element class for an accurate target.

    bs.find("h3", class_ = "s-item__title").string

    Scx24 C10 web Flatbed Axial Crawler 1/24 Scale Accessories Cooler Jerry Can

    Similarly, with the structure map done in figure 4, we can now extract information of products with Beautiful Soup.

    title = bs.find("h3", class_ = "s-item__title").string
cover = bs.find("img", class_ = "s-item__image-img").get("src")
price = bs.find("span", class_ = "s-item__price").string
print({"title": title, "cover": cover, "price": price})

    Multiple Item Extraction#

    Beautiful Soup provides a .find_all() function that behaves as a multi-target version of .find(). With this function, it is simple to accomplish what has been done to the whole HTML source with regular expressions in section Analyze Page Structures. In other words, we are about to extract all items on the page simultaneously.

    titles = bs.find_all("h3", class_ = "s-item__title")
covers = bs.find_all("img", class_ = "s-item__image-img")
prices = bs.find_all("span", class_ = "s-item__price")
items = []
for i in range(1, len(titles)):
    item = {
        "title": titles[i].string,
        "cover": covers[i].get("src"),
        "price": prices[i].string
    }
    items.append(item)
print(items)
